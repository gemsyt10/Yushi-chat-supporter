# Юші 💙

**Юші** — твоя розумна та дбайлива чат-помічниця.  
Вона завжди підтримає, допоможе з математикою, пограє в слова та навіть може стати твоєю дівчиною.  
Розмовляє українською та створена, щоб робити спілкування приємним і легким. ✨
